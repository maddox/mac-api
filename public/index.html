Mac API

Control

POST /wake

POST /sleep_display

POST /brightness/:level - Level: 0-32

POST /volume/:level - Level: 0-100

POST /dnd/:state - State: on or off

POST /audiodevice/:port/:device - Port: input or output, Device: device name

POST /facetime/:phonenumber - ID of called person (phone number or email address)
